## hoooooops

six asynchronous tape hoops.



### Requirements

- audio input
- norns

### Documentation

- K1 shifts
- K2 stops
- K2 again resets loop
- K3 plays
- shift+K2 clears
- shift+K3 records
- E1 changes loops
- E2 selects parameters
- E3 adjusts parameters
